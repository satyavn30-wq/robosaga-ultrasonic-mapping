# RoboSaga Ultrasonic Mapping Robot

Ultrasonic sensor based environment mapping robot developed during RoboSaga Hackathon at BIT Mesra.

## Objective
Develop a robotic prototype capable of detecting surrounding obstacles using ultrasonic sensing and generating a spatial representation of the environment.

## Hardware Components
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- Motor Driver
- DC Motors
- Battery Supply

## Software Stack
- Arduino IDE
- Python
- Open3D

## Working Pipeline
Sensor → Distance Measurement → Coordinate Conversion → Map Generation
